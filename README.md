## 🍲 Recetas de Comida Caliente (Hot Food)

* **Beef Stew (Estofado de ternera):** Olla (con agua) + Carne + Patata + Zanahoria.
* **Chicken Soup (Sopa de pollo):** Olla (con agua) + Pollo + Arroz + Verdura.
* **Goulash:** Olla (con agua) + Carne + Cebolla + Pimiento.
* **Pumpkin Soup (Sopa de calabaza):** Olla (con agua) + Calabaza + Grasa (Lard) + Sal.

## 🍰 Postres y Dulces

* **Apple Pie (Tarta de manzana):** Caja + Manzanas + Harina + Azúcar.
* **Cereals (Cereales):** Caja + Maíz + Azúcar + Leche.
* **Fruit Salad:** Caja + Manzana + Pera + Ciruela + Plátano.

---

> **Nota:** Recuerda que para la mayoría de las recetas que usan la **Olla**, si no se especifica "Water", se deben cocinar en seco para que el resultado sea el correcto según el mod.

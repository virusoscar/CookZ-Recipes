# 🍳 Guía de Recetas - Mod CookZ (DayZ)

Esta es una guía rápida de crafteo y cocina para el mod **CookZ**, diseñada para tener a mano todos los ingredientes y pasos necesarios.

## 📦 Recetas en Caja (Box Recipes)

Para estas recetas, generalmente se combina una **Caja Vacía** con los ingredientes dentro de una **Olla o Sartén** (sin agua o con agua según se indique) y se aplica fuego.

### Combinaciones con Patata Asada (Baked Potato)
* **Baked Potato + Meat:** Olla (sin agua) + 2 Patatas + 2 Carne (cualquiera) + Queso + Caja.
* **Baked Potato + Seafood:** Olla (sin agua) + 2 Patatas + Pescado + Queso + Caja.
* **Baked Potato + Vegetables:** Olla (sin agua) + 2 Patatas + Calabacín + Pimiento + Tomate + Queso + Caja.
* **Baked Potato + Beans:** Olla (sin agua) + 2 Patatas + Lata de Alubias + Queso + Caja.

### Platos Clásicos
* **Bangers and Mash:** Olla (**CON AGUA**) + Salchichas + 2 Patatas + Caja.
* **Biff a la Nelson:** Olla (sin agua) + 2 Carne + 2 Patatas + Manteca (Lard) + Caja.
* **Chicken Cordon Bleu:** Olla (sin agua) + Jamón o Pollo + 2 trozos de Queso + Caja.
* **Currywurst:** Sartén + Salchicha + Patata + Tomate + Caja.
* **Fish and Chips:** Sartén + Pescado + Carne + Patata + Caja.
* **Fish Sticks (Nuevo):** Sartén + Jamón o Pescado + Caja.

---

## 🍲 Recetas de Comida Caliente (Hot Food)

* **Beef Stew (Estofado de ternera):** Olla (con agua) + Carne + Patata + Zanahoria.
* **Chicken Soup (Sopa de pollo):** Olla (con agua) + Pollo + Arroz + Verdura.
* **Goulash:** Olla (con agua) + Carne + Cebolla + Pimiento.
* **Pumpkin Soup (Sopa de calabaza):** Olla (con agua) + Calabaza + Grasa (Lard) + Sal.

---

## 🍰 Postres y Dulces

* **Apple Pie (Tarta de manzana):** Caja + Manzanas + Harina + Azúcar.
* **Cereals (Cereales):** Caja + Maíz + Azúcar + Leche.
* **Fruit Salad:** Caja + Manzana + Pera + Ciruela + Plátano.

---

> **Nota:** Recuerda que para la mayoría de las recetas que usan la **Olla**, si no se especifica "Water", se deben cocinar en seco para que el resultado sea el correcto según el mod.
